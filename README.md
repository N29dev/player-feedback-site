# EgoIndex Player Feedback Website

Static HTML demo for a Roblox/BLR player feedback website.

## How to run
Open `index.html` in your browser.

## How to deploy on Vercel
1. Upload these files to your GitHub repo.
2. Import the repo in Vercel.
3. For a static HTML site, no build command is needed.
